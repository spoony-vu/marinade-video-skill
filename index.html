Setup
Start from the HyperFrames scaffold

The scaffold at assets/videos/_starter/hyperframes/ in marinade-finance/internal-docs ships pre-wired with 5 scenes, GSAP timeline, and brand tokens. Copy it, fill the blanks.

1

Clone the internal-docs repo

The starter, snippets, and playbook all live there.

gh repo clone marinade-finance/internal-docs cd internal-docs
2

Copy the starter to your working directory

Rename to your video's name. Each video lives in its own folder.

cp -R assets/videos/_starter/hyperframes/ ~/my-video/ cd ~/my-video/
3

Fill SCRIPT.md — the beat sheet

Five beats: Hook, Problem, Promise, Proof, CTA. Write one line per beat before touching the HTML.

4

Edit index.html — one scene at a time

Each scene is a <div class="clip"> with data-start, data-duration, data-track-index. Swap copy, swap tokens, adjust eases.

5

Preview in the browser

Hot-reload preview. Scrub the timeline, check pacing, validate transitions.

npx hyperframes preview
Structure
The 5-beat skeleton

All three production Marinade videos share this shape. Total length: 15–22 seconds. 18s is the sweet spot. Never exceed 22s without a reason.

0–2s
Hook
In-motion-on-open. Stops the scroll. Not a logo card.
2–6s
Problem
Hint, don't explain. One line, big type. The user's current state.
6–14s
Promise
Product glimpse. Show, don't list. UI cards, token bloom, counters.
14–18s
Proof
One stat or a sliver of UI doing something. Or a mantra.
18–22s
CTA
Wordmark + caption. Status verb. Period termination.

1920×1080, 30 fps

16:9 is the default — works on X, LinkedIn, web hero, blog embed. Use 1080×1920 only for stories/reels.

Hard transition rule

Never use exit animations except on the final scene. The transition IS the exit. The outgoing scene's content must be fully visible when the transition starts.

One per video, max

Per-character kinetic reveal: once per piece. PP Editorial New italic: one word per piece. Wipe transitions: one max.

Motion snippets
Production-extracted GSAP patterns

Five reusable patterns pulled from marinade-rewards-video. Copy the code into your composition at the right scene, adjust copy and timing.

1
Cursor click + pill ripple
Interaction

Cursor arcs in with power4.inOut, settles on target, pulses, ripples on click. Source widget gets a 1.012 yoyo — subtle, not cartoonish.

// Cursor arcs in, then click + ripple tl.fromTo("#cursor", { x: 1400, y: 800, opacity: 0 }, { x: 920, y: 510, opacity: 1, duration: 0.7, ease: "power4.inOut" }, 3.7); // Click bump on target (subtle, 1.012 max) tl.to("#target-pill", { scale: 1.012, duration: 0.12, ease: "power2.out" }, 4.4); tl.to("#target-pill", { scale: 1.0, duration: 0.12, ease: "power2.in" }, 4.52); // Pill ripple tl.to("#target-pill .ripple", { scale: 7.5, opacity: 0, duration: 0.5, ease: "power2.out" }, 4.4);
Gotcha. Don't compress the cursor itself — pros don't press cursors like buttons. Keep cursor scale ~1.0 throughout the arc. Ripple is on the pill, not the cursor.
2
Counter-up
Numbers

Single number counts from 0 to target with snap: { textContent }. Always tabular-nums so digits don't shift width.

// Count 0 → 8.16 over 1.4s, snap to 0.01 tl.to("#apy-value", { textContent: 8.16, duration: 1.4, ease: "power2.out", snap: { textContent: 0.01 }, onUpdate: function () { // Format to 2 decimals const target = this.targets()[0]; target.textContent = Number(target.textContent).toFixed(2); }, }, 9.0);
Gotcha. font-variant-numeric: tabular-nums is mandatory — without it, "8.16" renders narrower than "0.00" and the number slides left as it counts. Linear easing reads mechanical; use power2.out.
3
Per-character kinetic reveal
Headline

Headline letters fly in individually with back.out(2.2) and 45ms stagger. Reserve for ONE headline per video — overuse kills the moment.

tl.from("#rev-target .ichar", { y: 60, opacity: 0, scale: 0.7, duration: 0.7, ease: "back.out(2.2)", stagger: 0.045, }, 4.0); // Helper — wraps each char in a .ichar span before timeline construction function splitChars(el) { const text = el.textContent; el.innerHTML = ""; for (const ch of text) { const span = document.createElement("span"); span.className = "ichar"; span.textContent = ch === " " ? "\u00A0" : ch; el.appendChild(span); } }
Gotcha. Spaces must be \u00A0 — regular spaces collapse in display: inline-block children. Stagger sweet spot is 45ms: 30 reads frantic, 60 drags.
4
Token bloom from focal point
Catalog reveal

8–12 token icons bloom in 2–3 rows. Alternate rotation direction per row (−25 / +25 / −25). After bloom, settle with a yoyo y: -6 drift.

// Row 0 — rotate from -25 to 0 tl.from('.row[data-row="0"] .token', { scale: 0.7, opacity: 0, y: 16, rotation: -25, duration: 0.55, ease: "back.out(1.8)", stagger: 0.06 }, 9.6); // Row 1 — rotate from +25 to 0 (alternate direction) tl.from('.row[data-row="1"] .token', { scale: 0.7, opacity: 0, y: 16, rotation: 25, duration: 0.55, ease: "back.out(1.8)", stagger: 0.06 }, 10.1); // Settle drift — random stagger, breathes like one organism tl.to(".token", { y: -6, duration: 1.4, yoyo: true, repeat: 1, ease: "sine.inOut", stagger: { each: 0.04, from: "random" } }, 11.4);
Gotcha. Compress aggressively — white-background scenes drag at video speeds. Total bloom should take ~1.5s, not 3s. Random stagger on settle reads like a breath; sequential reads like a wave.
5
Blur crossfade transition
Transition

Default transition for Marinade videos. Premium, no visual debris. ~0.4–0.6s with sine.inOut. HyperFrames handles scene visibility automatically — you do NOT animate the scene wrapper opacity.

// 0.3s blur up, 0.3s blur down tl.to("#blur-overlay .overlay-fill", { opacity: 1, backdropFilter: "blur(18px)", duration: 0.3, ease: "sine.inOut" }, 3.0); tl.to("#blur-overlay .overlay-fill", { opacity: 0, backdropFilter: "blur(0px)", duration: 0.3, ease: "sine.inOut" }, 3.3);
Hard rule. Never use exit animations except on the final scene. The transition IS the exit. If you tl.to(opacity: 0) the scene before the transition, you get an empty frame.

Sentence case

ALL CAPS only for acronyms: SOL, USDC, USDG, MNDE, APY, TVL, MEV, SAM, PSR.

One italic word, max

PP Editorial New italic on a single accent word. Italic reads as emphasis only when scarce.

Tabular numbers

font-variant-numeric: tabular-nums on every dynamic number — APY, TVL, prices, dates.

Voice
Fintech clarity, DeFi substance

Sound like Mercury or Plaid, not like a CEX or a yield farm. Plain verbs, defensible numbers, no decoration. Every sentence should be readable by someone who has never used Solana, while every fact would survive scrutiny by someone who has.

Do

"Deposit USDC, earn yield, withdraw anytime."

"Marinade Select is live."

"Stake mSOL via btcSOL to earn zBTC."

"Up to 14% APY across 100+ validators."

"Coming soon." / "Early access." / "Live now."

Don't

"Marinade Native surpasses 1.8M SOL stake!"

"Unlock asset flexibility — the future of yield"

"best-in-market staking rewards"

"We're excited to announce…"

"Stay up to date on how the Chefs are cooking up new recipes"

No exclamation marks

Period termination always. If a line loses energy without "!", rewrite the line.

No em-dashes

Replace with periods, commas, or colons. Hyphens in compound words (non-custodial) are fine.

Numbers up front

"8.16% APY" not "high APY". "1.8M SOL staked" not "leading TVL". Mechanism named, not hyped.

No banned vocab

Skip: Chefs, kitchen, secret sauce, unlock, supercharge, top-tier, best-in-market, world-class, next-gen, trustless.

Status verbs

Coming soon / Early access / Live now. Not "now live" or "available today" for unshipped products.

Acronyms on first use

Spell out once: "maximal extractable value (MEV)". After that, the acronym is fine.

Audio
Music via ElevenLabs, paced to the video

Prompt-driven music generation. Generate the track first when you have a clear mood; match the video's pacing to the music. Or build silent, then commission a track that fits the cuts you already made.

1

Prompt ElevenLabs Music with a tight brief

Name the genre, instrumentation, tempo, and exact duration. The shorter the prompt, the closer the model lands.

warm lo-fi beat, mellow piano, soft kick, no vocals, 90 bpm, 22 seconds, fades out at the end
2

Hand the audio to Claude with the pacing ask

Drop the mp3 into the project's assets/audio/ directory and tell Claude to match scene transitions to the music's downbeats. Ask for the beat-grid mapping before any code changes.

read assets/audio/music.mp3, find the downbeats, then adjust the data-start times on each scene so transitions land on beat boundaries. Keep total duration at 22s.
3

Wire the audio into index.html

Audio is a separate <audio> element with its own data-track-index — never the audio track of a <video>. HyperFrames syncs playback to the timeline.

<audio id="music" data-start="0" data-duration="22" data-track-index="3" src="assets/audio/music.mp3" data-volume="0.85" ></audio>
Render
From preview to shareable MP4

The HyperFrames CLI covers the full lifecycle. Lint first, validate contrast, inspect for visual overflow, then render.

# Live preview in the browser npx hyperframes preview # Lint — check timing, attributes, IDs, overlap rules npx hyperframes lint --verbose # WCAG contrast audit at 5 timestamps npx hyperframes validate # Visual layout audit — cropped text, overlapping elements npx hyperframes inspect # Render to MP4 (headless Chrome + ffmpeg) npx hyperframes render --output out/video.mp4 # Shareable preview link (optional) npx hyperframes publish

Lint first

Catches overlapping tweens, missing data attributes, exit animations on non-final scenes.

Validate contrast

Body text on white must hit WCAG AA. brand-teal on white is for 18pt+ only.

Inspect for overflow

Catches headlines wrapping unexpectedly, tokens bleeding off-frame, captions clipping.

Pre-ship
Final checklist before publishing

Every item must pass. Cut corners and the brand drifts. Read silently, then watch the video muted at full speed — if it doesn't land without sound, the motion is the problem.

Identity

  • All colors from :root CSS vars
  • No hardcoded hex in inline styles
  • PP Mori on all body/headlines
  • PP Editorial New italic on ≤1 accent word
  • tabular-nums on every dynamic number
  • Logo variant matches background

Voice

  • No exclamation marks anywhere
  • No em-dashes (—) in copy
  • No banned vocabulary
  • Sentence case, acronyms keep casing
  • Status verb correct (Coming soon / Early access / Live now)
  • Acronyms expanded on first use

Motion

  • At least 3 different eases per scene
  • Per-character kinetic reveal on ≤1 headline
  • Money frame held 0.8s+ with internal drift
  • No exit animations except final scene
  • Cursor arcs, doesn't press
  • Catalog bloom compressed to ~1.5s

Build

  • npx hyperframes lint passes
  • npx hyperframes inspect clean
  • npx hyperframes validate clean
  • Audio on separate <audio> track
  • Total length 15–22s
  • Mute test: reads at full speed without sound
